<objective>
Build the graph editor at `/roadmaps/[id]` (REQ-admin-graph-editor): a three-zone layout (toolbar + React Flow canvas + node inventory) supporting node create (right-click on canvas / Add button), drag-to-place from inventory, edit via side panel, unplace-vs-permanent-delete semantics, a dirty-state Save Graph button, and an unsaved-changes navigation guard.
