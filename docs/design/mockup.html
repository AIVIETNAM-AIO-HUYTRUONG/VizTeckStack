Lesson · HTML

Semantic HTML

Semantic HTML means using the right HTML element for the right job. Instead of wrapping everything in <div>, semantic elements like <article>, <nav>, and <section> communicate meaning to both browsers and assistive technologies.

Why it matters

Screen readers rely on semantic structure to announce content correctly. Search engines use it to understand page hierarchy. Other developers use it to navigate your markup without reading every line.

<!-- Non-semantic -->
<div class="header">
  <div class="nav">...</div>
</div>

<!-- Semantic -->
<header>
  <nav>...</nav>
</header>

Key elements

Use <main> for the primary page content, <aside> for tangential content, and <figure> with <figcaption> for images that need captions.
